# Spring-React-Practice

### (Backend) Board url
* 리스트 - (GET) /api/board/
* 등록 - (POST) /api/board/
* 조회 - (GET) /api/board/{bno}
* 삭제 - (DELETE) /api/board/{bno}
* 수정 - (PUT) /api/board/{bno}

### (Frontend) Board url
* 리스트 - /board
* 조회 - /board/:bno
* 등록 - /board/register
